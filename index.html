PackMan

This is a Pack-man animation where four pictures are linked objects, then iterated to simulate mouth movement. The movement continues within the screen boundaries. When it reaches the Boundary it reverses course.

Future improvements:

  • Add a maze backgound
  • Add directional commands for Pack-Man
  • Add content for Pack-man to consume

    • To run code, download source material and run index.html

    Eyes Animation

    A simple animation using hyper text markup, Cascading styles sheets and JavaScript. Where a pair of animated eye tracks a mouse cursor across the screen. When the cursor moves the eye balls follow it the cursors direction.

    Future improvements:

  • Adding a Face
  • Add different eye colors
  • Add intermittent eye blincking

    • To run code, download source material and run index.html

    Real Time Bus Tracking

    This applications uses mapBox APIs to display maps, directions and animations. The real Time bus tracker follows markers points. Showing several bus stops from MIT to Harvard, animating each stop as an overlay in a three-dimensional map.

    Future improvements:

  • Add latitude and longitude input
  • add multiple stops input
  • add destination sequence changes
